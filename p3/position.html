Position : STATIC

Position static digunakan untuk element menjadi statis secara default. Ekemen akan mengikuti posisi berdasarkan alur normal web. Elemen tidak dipengaruhi oleh properti seperto top, bottom, left, dan right

Elemen div ini memiliki posisi "static"

Position : RELATIVE

Elemen dengan posisi relaive diposisikan relatif terhadap posisi normal

Elemen div ini memiliki posisi relative

Position : FIXED

Sebuah elemen dengan posisi fixed; diposisikan relatif terhadap viewport yang berarti selalu tetap di tempat yang sama bahkan jika halaman di-scroll:

Elemen div ini memiliki posisi fixed

Position : ABSOLITE

Sebuah elemen dengan posisi : absolute diposisikan relatif terhadap elemen lain yang terdekat, bukan relatif terhadap viewport

Elemen divini memiliki posisi "relative"
Elemen div ini memiliki posisi "absolute"
